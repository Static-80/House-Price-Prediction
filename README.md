# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview

This project predicts house prices using the Linear Regression algorithm. The dataset contains various house features such as living area, number of bedrooms, bathrooms, garage capacity, and overall quality. The goal is to build a machine learning model that estimates house prices based on these features.

---

## 🎯 Objective

The objective of this project is to:

- Perform Exploratory Data Analysis (EDA)
- Clean and preprocess the dataset
- Handle missing values
- Select important features
- Train a Linear Regression model
- Evaluate model performance using regression metrics
- Visualize predictions and residuals
- Interpret feature importance using model coefficients

---

## 📂 Dataset

**Dataset:** House Prices - Advanced Regression Techniques

The dataset contains information about residential homes and their sale prices.

Main features used:

- GrLivArea (Ground Living Area)
- BedroomAbvGr (Number of Bedrooms)
- FullBath (Number of Bathrooms)
- GarageCars (Garage Capacity)
- OverallQual (Overall Quality)

**Target Variable:**

- SalePrice
